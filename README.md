Vizualizacija
=============

About
-----

Vizualizacija is a simple logic diagram drawer. It was a student project done
during the course Basics of Java Programming Language. The team was composed
of 5 students:
- Igor Belša
- Boran Car
- Jurica Cerovec
- Robert Fitoš
- Ivan Krišto
